# Fix: Bottom tab icons don't adapt to light theme

## Problem
`NativeTabs` icons use `renderingMode="template"` but no `tintColor` is set, so icons stay dark regardless of theme.

## Fix
**File:** `src/components/app-tabs.tsx`

Add `tintColor={colors.text}` prop to the `NativeTabs` component (line 26).

This single prop change will make template-rendered icons tint to black in light mode and white in dark mode.

## Verification
- Toggle to Light in settings → tab icons should turn black
- Toggle to Dark in settings → tab icons should turn white
